# 3D Анимированные кубы - LOADING

Интерактивная 3D анимация с использованием CSS Transformations и JavaScript.

## Особенности
- 3D кубы с подсветкой
- Смена цветов в реальном времени
- Переключение темы (темная/светлая)
- Полностью адаптивный дизайн

## Технологии
- HTML5, CSS3 (3D Transforms, Animations)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts
